## Getting Started

### Prerequisites

- Go (latest version recommended). Download from: https://golang.org/dl/

### Installation

1. Clone this repository:
   ```sh
   git clone https://github.com/mrrajput04/go-basic-servers.git
   cd go-basic-servers
   ```

2. Navigate to the server app directory:
   ```sh
   cd basic-server-app
   ```

3. Run the server:
   ```sh
   go run main.go
   ```
